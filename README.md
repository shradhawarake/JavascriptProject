# JavascriptProject

A small collection of JavaScript examples and Playwright exercises used while learning browser automation.

## Contents
- Chapter_01_Basics: simple JS examples and small scripts

## Prerequisites
- Node.js (recommended v18+)
- npm or yarn

## Quick start
1. Open a terminal in the project root.
2. Run a simple script with Node:

```powershell
cd "C:\Users\shrad\OneDrive\Desktop\playwright course\JavasriptProject"
node Chapter_01_Basics\01_Basic.js
```

## Playwright tests
- If you add Playwright tests, install Playwright and run them:

```bash
npm init -y
npm i -D @playwright/test
npx playwright install
npx playwright test
```
